## [1.0.0] - 2026-05-15

### Added
- Initial public release of ShadowForge DevTools.
- In-game developer panel accessible via `/devtools` command.
- Entity inspector (vehicles, peds, props) with property breakdown.
- One-click coordinate copy to clipboard.
- Target zone builder with visual placement.
- Prop placement and snippet generation.
- Framework adapters for Standalone, Qbox, QBCore, and ESX.
- `ox_lib` integration for menus and notifications.
